# Task Management System (Console)

Тестовое задание на позицию Java-разработчик (Стажировка) в компанию Aveelon. 

## Описание программы
Мини-приложение для управления списком задач через консольный интерфейс.

Основные функции: 
 Добавление задач: Создание новой задачи с уникальным UUID. 
 Просмотр: Вывод списка всех задач в структурированном виде. 
 Управление статусом:** Возможность менять статус на NEW, IN_PROGRESS или DONE. 
 Удаление: Удаление задачи по её идентификатору.
 Сохранение данных:** Автоматическое сохранение в текстовый файл (`tasks.txt`) при выходе и загрузка при старте.

## Инструкция по запуску
1.  Убедитесь, что у вас установлена Java (JDK 8 или выше).
2.  Скомпилируйте файлы из папки `src`.
3.  Запустите класс `Main`.
4.  Для управления используйте цифровые команды (1-4) в консоли. Для сохранения и выхода введите `0`.
